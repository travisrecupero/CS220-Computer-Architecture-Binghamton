EX2

The mystery function takes in an argument and give a different number. 
When we run objdump on the mystery object file, we see that the assembly code
makes use of an 'and'. In this example we see:
		  
			'and   $0x7fffff,%eax'
			
....so the extended accumulator register (%eax) is storing the mask.
